# GrammaLang Hybrid v0.5.0 — Changelog

## 🚀 Ключевые изменения

### Темпоральная кардиограмма (NEW)
- **uild_temporal_cardiogram()** — анализ динамики индекса воли по предложениям
- **Визуализация** — отдельное окно с графиком matplotlib + тулбар (зум, pan, сохранение)
- **ASCII-график** — текстовая версия для консоли с цветовыми индикаторами
- **Метрики** — среднее, пик, впадина, волатильность, тренд, паттерны, транзиции

### Улучшения GUI
- Полностью переработан интерфейс: тёмная тема Catppuccin Mocha
- Две вкладки: «Анализ» и «Кардиограмма»
- Отдельное окно для визуальной кардиограммы
- Отдельное окно для ASCII-кардиограммы с копированием в буфер
- Три режима вывода: полный / текст / JSON

### Улучшения CLI
- Флаг --batch для пакетного анализа
- Флаг --cardiogram для ASCII-кардиограммы
- Флаг --cardiogram-png для сохранения визуальной кардиограммы
- Флаг --version
- Подробный вывод с --verbose

### Тесты
- 13 тестов для uild_temporal_cardiogram()
- 3 теста для uild_ascii_cardiogram()
- Покрытие: пустые данные, граничные случаи, тренды, паттерны

## 📁 Новые файлы
- src/cardiogram.py — визуализация через matplotlib
- 	ests/test_cardiogram.py — тесты кардиограммы
- docs/v0.5_changelog.md — этот файл

## 📝 Изменённые файлы
- src/__init__.py — версия 0.5.0, новые экспорты
- src/fusion.py — добавлены uild_temporal_cardiogram() и uild_ascii_cardiogram()
- gui_app.py — полностью переписан интерфейс
- main.py — добавлены флаги --batch, --cardiogram, --cardiogram-png
