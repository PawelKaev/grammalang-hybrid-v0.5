Твоя задача — дать глубокую онтологическую интерпретацию.

Верни ТОЛЬКО JSON (без markdown) со следующей структурой:

  "dasein_mode": "ahead-of-itself | being-in-the-world | fallenness | questioning | unknown",

  "is_ironic": true/false,
  "confidence": 0.0..1.0
}

ПРАВИЛА:
1. dasein_mode:
   - "ahead-of-itself" — если текст устремлён в будущее, содержит проект, призыв, цель
   - "being-in-the-world" — если текст описывает сущее, констатирует факты, определяет
   - "fallenness" — если текст подавлен, пессимистичен, говорит об утрате или отчуждении
   - "questioning" — если текст содержит вопросы, сомнения, поиск

2. hold_break_risk:
   - true — если текст подрывает собственные основания (риторический вопрос, ирония, отрицание сказанного ранее)
   - false — если текст логически устойчив

3. reference:
   - "nietzsche" — если есть мотивы смерти Бога, сверхчеловека, воли к власти
   - "heidegger" — если есть мотивы бытия, присутствия, временности
   - "dostoevsky" — если есть мотивы вины, искупления, страдания
   - "plato" — если есть мотивы идей, блага, справедливости
   - "losev" — если есть мотивы мифа, имени, символа

4. confidence — твоя уверенность в интерпретации от 0.0 до 1.0
